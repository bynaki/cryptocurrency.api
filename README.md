# Cryptocurrency API


## Setting Config File

* rename `config.base.json` to `config.json`
* correct `config.json`


## Test

```bash
yarn test
```


## Publish

```bash
yarn build.tsc      # 일단 빌드한다
yarn login          # 로그인
yarn publish        # 프로젝트의 루트 위치에서, npm에 배포 등록한다
```
